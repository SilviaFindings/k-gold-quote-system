## 🚀 快速开始

### 克隆项目
```bash
git clone https://github.com/SilviaFindings/k-gold-quote-system.git
cd k-gold-quote-system
```

### 安装依赖
```bash
pnpm install
```

### 启动开发服务器
```bash
pnpm dev
```
访问：http://localhost:5000
