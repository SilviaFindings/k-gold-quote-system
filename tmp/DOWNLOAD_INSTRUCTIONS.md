## 🚀 本地运行步骤（超简单）

### 第1步：安装 Node.js

访问 https://nodejs.org/ 下载并安装 Node.js（推荐 LTS 版本）

**验证安装：**
打开命令行/终端，输入：
```bash
node --version
# 应该显示 v18.x.x 或更高
```

### 第2步：安装 pnpm（推荐）

在命令行中输入：
```bash
npm install -g pnpm
```

或者直接使用 npm（如果不想安装 pnpm）

### 第3步：解压并进入项目目录

```bash
# 解压文件到您选择的目录
# 然后进入项目目录
cd k-gold-pricing-system
```

### 第4步：安装依赖

```bash
pnpm install
```
等待安装完成（可能需要 1-3 分钟）

### 第5步：启动应用

```bash
pnpm dev
```

看到类似以下提示表示启动成功：
```
✓ Ready in 2.5s
○ Local: http://localhost:5000
```

### 第6步：使用应用

打开浏览器访问：http://localhost:5000

🎉 现在您可以开始使用报价表了！

## 💡 首次使用提示

### 导入数据

如果您有 Excel 数据，可以：
1. 点击"Excel批量导入"按钮
2. 选择您的 Excel 文件
3. 设置导入选项（是否导入重量、人工成本等）
4. 点击"开始导入"

### 手动录入

1. 选择产品分类
2. 填写产品信息（货号、名称、规格、重量、人工成本）
3. 选择金种（14K 或 18K）
4. 系统会自动计算价格

### 备份数据

⚠️ **重要**：数据保存在浏览器中，清除浏览器数据会丢失！

定期点击"数据管理"中的"导出备份"按钮下载数据备份。

## 🔧 常见问题

### Q: 安装依赖时报错
**A:** 检查 Node.js 版本（需 18+），或尝试使用 `npm install`

### Q: 启动后页面无法访问
**A:** 检查端口 5000 是否被占用，或查看命令行错误信息

### Q: Excel 导入失败
**A:** 确保 Excel 文件包含"货号"和"名称"列

### Q: 修改代码后页面不更新
**A:** Next.js 支持热更新，保存文件后自动刷新。如未生效，按 Ctrl+C 停止后重新运行 `pnpm dev`
