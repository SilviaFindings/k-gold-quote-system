## 常见问题

### Q: npm install 报错？
A: 确保安装了Node.js 18或更高版本。访问 https://nodejs.org/ 下载。

### Q: 找不到page.tsx文件？
A: 等我将项目推送到GitHub，然后直接克隆仓库：`git clone https://github.com/SilviaFindings/k-gold-quote-system.git`

### Q: 如何导入数据？
A: 系统中有"导入Excel"功能，支持批量导入产品数据。
